# Telegram Currency Bot
This Bot Is Reference Form [Blurdylan](https://github.com/blurdylan/telegram-currency-converter-bot/tree/tutorial)


## Dependecies
- Node JS
- Telegraf
- Axios
- Currency API
- Nodemon


## Installation
```bash
npm install
```
## Run Development
```bash
npm run dev
```
